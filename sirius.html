Current Commands

Command Name Description Syntax Permissions
Ban Bans a specific user from the server *ban @user ADMIN ONLY
Future Update Lists upcoming updates to Sirius Bot *futureupdate All users
Help Lists all commands and their syntaxes *help All users
Kick Kicks a specific user from the server *kick @user ADMIN ONLY
Purge Deletes a specified number of messages from the channel as long as they are within 14 days of being sent *purge # ADMIN ONLY
Server Info Shows information about the current server hosting Sirius Bot *serverinfo All users
Update Log Shows every update to Sirius and what it did *updatelog All users
